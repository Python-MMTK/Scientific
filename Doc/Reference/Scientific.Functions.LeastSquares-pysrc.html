 51      """General non-linear least-squares fit using the 
 52      X{Levenberg-Marquardt} algorithm and X{automatic differentiation}. 
 53   
 54      @param model: the function to be fitted. It will be called 
 55          with two parameters: the first is a tuple containing all fit 
 56          parameters, and the second is the first element of a data point (see 
 57          below). The return value must be a number.  Since automatic 
 58          differentiation is used to obtain the derivatives with respect to the 
 59          parameters, the function may only use the mathematical functions known 
 60          to the module FirstDerivatives. 
 61      @type param: callable 
 62   
 63      @param parameters: a tuple of initial values for the 
 64          fit parameters 
 65      @type parameters: C{tuple} of numbers 
 66   
 67      @param data: a list of data points to which the model 
 68          is to be fitted. Each data point is a tuple of length two or 
 69          three. Its first element specifies the independent variables 
 70          of the model. It is passed to the model function as its first 
 71          parameter, but not used in any other way. The second element 
 72          of each data point tuple is the number that the return value 
 73          of the model function is supposed to match as well as possible. 
 74          The third element (which defaults to 1.) is the statistical 
 75          variance of the data point, i.e. the inverse of its statistical 
 76          weight in the fitting procedure. 
 77      @type data: C{list} 
 78   
 79      @returns: a list containing the optimal parameter values 
 80          and the chi-squared value describing the quality of the fit 
 81      @rtype: C{(list, float)} 
 82      """ 
 83      n_param = len(parameters) 
 84      p = () 
 85      i = 0 
 86      for param in parameters: 
 87          p = p + (DerivVar(param, i),) 
 88          i = i + 1 
 89      id = N.identity(n_param) 
 90      l = 0.001 
 91      chi_sq, alpha = _chiSquare(model, p, data) 
 92      niter = 0 
 93      while 1: 
 94          delta = LA.solve_linear_equations(alpha+l*N.diagonal(alpha)*id, 
 95                                            -0.5*N.array(chi_sq[1])) 
 96          next_p = map(lambda a,b: a+b, p, delta) 
 97          next_chi_sq, next_alpha = _chiSquare(model, next_p, data) 
 98          if next_chi_sq > chi_sq: 
 99              l = 10.*l 
100          else: 
101              l = 0.1*l 
102              if chi_sq[0] - next_chi_sq[0] < stopping_limit: break 
103              p = next_p 
104              chi_sq = next_chi_sq 
105              alpha = next_alpha 
106          niter = niter + 1 
107          if max_iterations is not None and niter == max_iterations: 
108              raise IterationCountExceededError 
109      return [p[0] for p in next_p], next_chi_sq[0]
