   7  """ 
   8  Parsing and writing of Protein Data Bank (PDB) files 
   9   
  10  This module provides classes that represent PDB (Protein Data Bank) 
  11  files and configurations contained in PDB files. It provides access to 
  12  PDB files on two levels: low-level (line by line) and high-level 
  13  (chains, residues, and atoms). 
  14   
  15  Caution: The PDB file format has been heavily abused, and it is 
  16  probably impossible to write code that can deal with all variants 
  17  correctly. This modules tries to read the widest possible range of PDB 
  18  files, but gives priority to a correct interpretation of the PDB 
  19  format as defined by the Brookhaven National Laboratory. 
  20   
  21  A special problem are atom names. The PDB file format specifies that 
  22  the first two letters contain the right-justified chemical element 
  23  name. A later modification allowed the initial space in hydrogen names 
  24  to be replaced by a digit. Many programs ignore all this and treat the 
  25  name as an arbitrary left-justified four-character name. This makes it 
  26  difficult to extract the chemical element accurately; most programs 
  27  write the '"CA"' for C_alpha in such a way that it actually stands for 
  28  a calcium atom. For this reason a special element field has been added 
  29  later, but only few files use it. In the absence of an element field, 
  30  the code in this module attempts to guess the element using all information 
  31  available. 
  32   
  33  The low-level routines in this module do not try to deal with the atom 
  34  name problem; they return and expect four-character atom names 
  35  including spaces in the correct positions. The high-level routines use 
  36  atom names without leading or trailing spaces, but provide and use the 
  37  element field whenever possible. For output, they use the element 
  38  field to place the atom name correctly, and for input, they construct 
  39  the element field content from the atom name if no explicit element 
  40  field is found in the file. 
  41   
  42  Except where indicated, numerical values use the same units and 
  43  conventions as specified in the PDB format description. 
  44   
  45  Example:: 
  46   
  47    >>>conf = Structure('example.pdb') 
  48    >>>print conf 
  49    >>>for residue in conf.residues: 
  50    >>>    for atom in residue: 
  51    >>>        print atom
