 18      """ 
 19      Parallel task manager for shared-memory multiprocessor machines 
 20   
 21      This class provides a rather simple way to profit from 
 22      shared-memory multiprocessor machines by running several tasks 
 23      in parallel. The calling program decides how many execution threads 
 24      should run at any given time, and then feeds compute tasks to 
 25      the task manager, who runs them as soon as possible without exceeding 
 26      the maximum number of threads. 
 27   
 28      The major limitation of this approach lies in Python's Global 
 29      Interpreter Lock. This effectively means that no more than one 
 30      Python thread can run at the same time. Consequently, parallelization 
 31      can only be achieved if the tasks to be parallelized spend 
 32      significant time in C extension modules that release the Global 
 33      Interpreter Lock. 
 34      """
