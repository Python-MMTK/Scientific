11  """ 
12  Array indexing utility

14  This module provides a convenient method for constructing 
15  array indices algorithmically. It provides one importable object, 
16  L{index_expression}. 
17   
18  For any index combination, including slicing and axis insertion, 
19  C{a[indices]} is the same as C{a[index_expression[indices]]} for any 
20  array {a}. However, C{index_expression[indices]} can be used anywhere 
21  in Python code and returns a tuple of indexing objects that can be 
22  used in the construction of complex index expressions. 
23   
24  Sole restriction: Slices must be specified in the double-colon 
25  form, i.e. C{a[::]} is allowed, whereas C{a[:]} is not.
