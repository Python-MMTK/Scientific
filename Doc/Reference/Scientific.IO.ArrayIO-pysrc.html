  8  Basic support for I/O of one- and two-dimensional numerical arrays to and from plain text files 
  9   
 10  The text file format is very simple and used by many other programs as 
 11  well: 
 12   
 13    - each line corresponds to one row of the array 
 14   
 15    - the numbers within a line are separated by white space 
 16   
 17    - lines starting with # are ignored (comment lines) 
 18   
 19  An array containing only one line or one column is returned as a

 21  as one item per line. 
 22   
 23  Numbers in files to be read must conform to Python/C syntax.  For
