  8  """ 
  9  Potential energy functions with automatic gradient evaluation 
 10   
 11  This module offers two strategies for automagically calculating the 
 12  gradients (and optionally force constants) of a potential energy 
 13  function (or any other function of vectors, for that matter).  The 
 14  more convenient strategy is to create an object of the class 
 15  PotentialWithGradients. It takes a regular Python function object 
 16  defining the potential energy and is itself a callable object 
 17  returning the energy and its gradients with respect to all arguments 
 18  that are vectors. 
 19   
 20  Example:: 
 21   
 22    >>>def _harmonic(k,r1,r2): 
 23    >>>    dr = r2-r1 
 24    >>>    return k*dr*dr 
 25    >>>harmonic = PotentialWithGradients(_harmonic) 
 26    >>>energy, gradients = harmonic(1., Vector(0,3,1), Vector(1,2,0)) 
 27    >>>print energy, gradients 
 28   
 29  prints:: 
 30   
 31    >>>3.0 
 32    >>>[Vector(-2.0,2.0,2.0), Vector(2.0,-2.0,-2.0)] 
 33   
 34  The disadvantage of this procedure is that if one of the arguments is a 
 35  vector parameter, rather than a position, an unnecessary gradient will 
 36  be calculated. A more flexible method is to insert calls to two 
 37  function from this module into the definition of the energy 
 38  function. The first, DerivVectors(), is called to indicate which 
 39  vectors correspond to gradients, and the second, EnergyGradients(), 
 40  extracts energy and gradients from the result of the calculation. 
 41  The above example is therefore equivalent to:: 
 42   
 43    >>>def harmonic(k, r1, r2): 
 44    >>>    r1, r2 = DerivVectors(r1, r2) 
 45    >>>    dr = r2-r1 
 46    >>>    e = k*dr*dr 
 47    >>>    return EnergyGradients(e,2) 
 48   
 49  To include the force constant matrix, the above example has to be 
 50  modified as follows:: 
 51   
 52    >>>def _harmonic(k,r1,r2): 
 53    >>>    dr = r2-r1 
 54    >>>    return k*dr*dr 
 55    >>>harmonic = PotentialWithGradientsAndForceConstants(_harmonic) 
 56    >>>energy, gradients, force_constants = harmonic(1.,Vector(0,3,1),Vector(1,2,0)) 
 57    >>>print energy 
 58    >>>print gradients 
 59    >>>print force_constants 
 60   
 61  The force constants are returned as a nested list representing a 
 62  matrix. This can easily be converted to an array for further 
 63  processing if the numerical extensions to Python are available.
