 11   
 12  This module contains a Python interface to the Message Passing 
 13  Interface (MPI), and standardized library for message-passing parallel 
 14  computing. Please read an introduction to MPI before using this 
 15  module; some terms in the documentation do not make much sense unless 
 16  you understand the principles of MPI. 
 17   
 18  This module contains an object, 'world', which represents the 
 19  default communicator in MPI. This communicator can be used directly 
 20  for sending and receiving data, or other communicators can be 
 21  derived from it. 
 22   
 23  A number of global constants are also defined (L{max}, L{min}, L{prod}, 
 24  L{sum}, L{land}, L{lor}, L{lxor}, L{band}, L{bor}, L{bxor}, L{maxloc}), 
 25  and L{minloc}). They are used to specify the desired operator in calls to 
 26  the 'reduce' and 'allreduce' methods of the communicator objects.
