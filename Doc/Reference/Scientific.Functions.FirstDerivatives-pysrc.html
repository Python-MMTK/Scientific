  7  """ 
  8  Automatic differentiation for functions with any number of variables 
  9   
 10  Instances of the class DerivVar represent the values of a function and 
 11  its partial X{derivatives} with respect to a list of variables. All 
 12  common mathematical operations and functions are available for these 
 13  numbers. There is no restriction on the type of the numbers fed into 
 14  the code; it works for real and complex numbers as well as for any 
 15  Python type that implements the necessary operations. 
 16   
 17  This module is as far as possible compatible with the n-th order 
 18  derivatives module Derivatives. If only first-order derivatives 
 19  are required, this module is faster than the general one. 
 20   
 21  Example:: 
 22   
 23    print sin(DerivVar(2)) 
 24   
 25  produces the output:: 
 26   
 27    (0.909297426826, [-0.416146836547]) 
 28   
 29  The first number is the value of sin(2); the number in the following 
 30  list is the value of the derivative of sin(x) at x=2, i.e. cos(2). 
 31   
 32  When there is more than one variable, DerivVar must be called with 
 33  an integer second argument that specifies the number of the variable. 
 34   
 35  Example:: 
 36   
 37      >>>x = DerivVar(7., 0) 
 38      >>>y = DerivVar(42., 1) 
 39      >>>z = DerivVar(pi, 2) 
 40      >>>print (sqrt(pow(x,2)+pow(y,2)+pow(z,2))) 
 41   
 42      produces the output 
 43   
 44      >>>(42.6950770511, [0.163953328662, 0.98371997197, 0.0735820818365]) 
 45   
 46  The numbers in the list are the partial derivatives with respect 
 47  to x, y, and z, respectively. 
 48   
 49  Note: It doesn't make sense to use DerivVar with different values 
 50  for the same variable index in one calculation, but there is 
 51  no check for this. I.e.:: 
 52   
 53      >>>print DerivVar(3, 0)+DerivVar(5, 0) 
 54   
 55      produces 
 56   
 57      >>>(8, [2]) 
 58   
 59  but this result is meaningless. 
 60  """
